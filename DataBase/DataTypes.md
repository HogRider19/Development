**smallint** - знаковое двухбайтное целое
**int** - знаковое четырёхбайтное целое
**bigint** - знаковое целое из 8 байт

**real/float4** - число одинарной точности с плавающей точкой (4 байта)
**float8** - число двойной точности с плавающей точкой (8 байт)

**smallserial** - двухбайтное целое с инкрементированием
**serial** - четырёхбайтное целое с  инкрементированием
**bigserial** - восьмибайтное целое с инкрементированием

**char \[n]** - символьная строка фиксированной длины
**varchar \[n]** - символьная строка переменной длины 
**text** - символьная строка переменной длины

**boolean** - логическое значение (true/false)
**bytea** - двоичные данные ("массив байт")

**money** - денежная сумма
**numeric/decimal\[p, s]** - вещественное число заданной точности

**date** - календарная дата (год, месяц, день)
**time \[p]** - время суток (без часового пояса)

**json** - текстовые данные JSON 
**jsonb** - двоичные данные JSON, разобранные

**uuid** - универсальный уникальный идентификатор
