Для взаимодействия с сервером приложение Blazor WebAssembly использует класс HttpClient, который в реальности использует Fetch API браузера для отправки запросов. И для работы с HttpClient нам надо добавить данный объект в качестве сервиса в приложение

При добавлении HttpClient устанавливается свойство BaseAddress, которое определяет адрес веб-сервиса
